# Candidate Response Evaluation using AI

## Overview
This project evaluates candidate responses using AI and generates:
- Score
- Strengths
- Weaknesses
- Feedback

## Technologies Used
- HTML5
- CSS3
- JavaScript
- OpenRouter API
- Llama 3.1 8B Instruct

## Features
- AI-powered evaluation
- Empty response validation
- Short answer validation
- Structured JSON output

## How to Run
1. Clone the repository
2. Add your OpenRouter API key in script.js
3. Open index.html using Live Server
4. Enter question and answer
5. Click Evaluate Response

## Note
A valid OpenRouter API key is required to run this project.
